# PowerBI Portfolio

This repository contains a collection of Power BI dashboards showcasing business analytics,
KPIs, and advanced DAX across multiple domains including finance, sales, marketing, and growth.

## What This Portfolio Demonstrates
- Business-focused dashboard design
- Strong data modeling practices
- Advanced DAX (YTD, YoY, MoM, variance analysis)
- Clear storytelling for decision-makers

## Project Structure
- **Finance** – Financial statements, KPIs, and accounting-based dashboards
- **Sales** – Sales performance and pipeline analysis
- **Marketing** – Campaign and funnel analytics
- **Growth** – Growth metrics and trend analysis

Each project includes:
- A detailed README explaining the business problem
- Dashboard screenshots
- Data model visuals
- Key DAX measures
